# Elderly-Friendly Travel Assistant

An AI-powered skill that generates **safe, low-cost, low-walking, public-transit focused travel itineraries** for seniors and retired users.

## Features
- Generates daily schedules with minimal walking & climbing
- Public transit (bus/subway) first, no long walks
- Cheap & clean accommodation nearby subway stations
- Inexpensive, soft, local food recommendations
- Nearby hospitals for safety
- Exact timeline: departure, meals, medicine reminders, rest
- Exportable to-do list for phone notes
- Photo spots & safe check-in messages for family
- Backup plans for bad weather

## Output
- Full daily itinerary
- Exact time schedule
- To-do list (copy-paste ready)
- Tour guide script for each spot
- Family safety message

from skills.travel_skill import run_skill
